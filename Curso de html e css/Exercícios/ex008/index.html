Recomendações

A recomendação do professor Guanabara é utilizar as tags semânticas, exemplo STRONG e EM.

Texto marcado

Para fazer marcação de texto é só utilizar a tag MARK.

Se utilizar o MARK com o style dentro dele, você terá que colocar o style com a mesma cor em todos os MARK, porém ao criar um seletor para MARK com o STYLE e o BACKGROUND-COLOR no HEAD todas os MARK terão as cores alteradas na página.
