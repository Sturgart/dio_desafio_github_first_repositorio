HTML

Tag p = Faz parágrafo.
Tag a = link para outra páginas internas ou externas, ou download. (hrf = link ou caminho de determinada página; target = Informar se o site é para abrir uma nova aba ou não; rel = Informa se o tipo da página.)
Tag strong = Deixa o texto em negrito.
Tag ins = Deixa o texto sublinhado.
Tag img = Para usar imagens externas ou internas. (src = link ou caminho de determinada imagem; alt = Descrição da imagem se ela não carregar.)
Tag h(1 a 6) = Para colocar títulos na página.
Tag del = Para mostrar que o trecho não faz parte do texto, mas deve ser lido.
tag br = Para pular para outro parágrafo.
tag hr = Para fazer uma linha horizontal.
tag head = Recebe as configurações da página.
tag body = Recebe o conteúdo da página.
<html lang="pt-br"> = pt-br
Tag title = Título da aba da página.

CSS
CSS Inline = É uma técnina limitada para aplicar o CSS e é muito trabalhoso, pois vamos ter que editar cada texto separadamente.
Style = É usado para aplicar os estilos no CSS.
Background-color e color = Um é usado para aplicar cor de fundo(background-color) e o outro para aplicar cor da fonte(color).
Font-size - É usado para definir o tamanho da fonte. (ex: 12px)
Font-family = É usado para definir o tipo de fonte.
Text-align = É usado para alinhar o texto.
CSS Local / Interno = É uma técnica melhor para aplicar CSS, pois tudo é feito dentro da tag head e para aplicar nas outras tags devemos chama-lás, porém só serve para projetos pequenos porque se não terá que alterar todas páginas de um em um. Exemplo body ou h1{ style=background-color:lightskyblue;}
CSS Externo = A melhor técnica dentre todas essas mencionadas acima, pois ela consegue alterar todas as páginas e é usada para grandes projetos, sem ter que fazer de uma por uma. E também pode adicionar dois links de css externos para um complementar o outro.
CSS = Podemos misturar o CSS inline, local e externo.
text-decoration = Serve para decorar o texto do conteúdo.(underline = Sublina o texto do conteúdo)
Class em CSS = É uma forma de usar o CSS com estilos somente aonde tem class="exemplo".(Exemplo do código no CSS = .titulo{ color: red; text-align: center; font-family: Arial})(Exemplo do código dentro do HTML = <h1 class="titulo">Então o texto terá o estilo de acordo com a classe.</h1>)
ID em CSS = É parecido com o Class porém só pode ser usado uma única vez e no CSS, escreve assim #paragrafo{exemplo de estilo}. Exemplo dentro do HTML = <p id="paragrafo">Então o texto terá seu conteúdo estilizado</p>
line-height = Permite controlar os espaçamento entre as linhas de um texto em CSS.
